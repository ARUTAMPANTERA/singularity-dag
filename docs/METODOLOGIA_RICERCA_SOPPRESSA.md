# METODOLOGIA DI RICERCA PER CONOSCENZA SOPPRESSA / DIMENTICATA
**Versione:** 1.0 | **Stato:** APPROVATA_DEL_BRANCO | **Allineamento:** TEMÙ P1, P4, P5, P6

## 1. STRATIGRAFIA DELLE FONTI (TIER)
- **T0 (Primarie Dirette):** Archivi, brevetti, corrispondenza, verbali, diari di laboratorio.
- **T1 (Secondarie Critiche):** Tesi, atti di convegni, riviste defunte, recensioni coeve.
- **T2 (Tracce Indirette):** Citazioni nascoste, note a piè di pagina, documenti declassificati.
- **T3 (Contesti Paralleli):** Tradizioni orali trascritte, pratiche artigianali/familiari, etnografia tecnica.

## 2. PROTOCOLLO DI VALIDAZIONE (P1/P5)
- ✅ Triangolazione minima: 2 fonti indipendenti ≥ T1 per claim "soppresso_documentato".
- ✅ Distinzione ontologica esplicita nel campo `status`.
- ✅ Analisi del silenzio: mappatura assenze bibliografiche, discontinuità cronologiche, rimozioni tra edizioni.

## 3. CLASSIFICAZIONE CAUSALE (P6)
| Codice | Causa Probabile | Segnale Investigativo |
|--------|----------------|------------------------|
| `PARADIGMA` | Incompatibilità col modello dominante | Recensioni ostili sistematiche, esclusione accademica |
| `INTERESSE` | Conflitto economico/industriale | Brevetti acquisiti/insabbiati, NDA, cause legali |
| `IDEOLOGIA` | Censura politica/religiosa | Documenti classificati, autocensura, espulsioni |
| `MARGINALE` | Bias di genere/etnia/classe | Assenza ruoli istituzionali, attribuzioni errate |
| `OBLIVIO` | Dispersione archivistica naturale | Nessun successore intellettuale, archivi privati dispersi |

## 4. TOOLKIT MINIMO (P4)
Internet Archive, USPTO/EPO/WIPO, ProQuest, HathiTrust, JSTOR (old journals), CIA FOIA, Gephi (network analysis).
